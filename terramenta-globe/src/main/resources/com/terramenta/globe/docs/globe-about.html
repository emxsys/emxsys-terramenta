Using the Globe

The Globe window is a tool for visualizing the virtual earth, and it is an input device for selecting geographical areas to be analyzed. 

View Controls

Several buttons are displayed in the corner of the viewer which are used for controlling the view angles, panning and zooming.

Pan Navigation

  • Mouse
    Click on the globe to pan the virtual earth to the clicked location; Left drag-click to scroll the globe to the desired location.
  • Keyboard
    Use the [Left/Right] and [Up/Down] arrow keys to move the globe.
    Use the [ALT]+[Left/Right] or [Left/Right] keys to move around slowly.

Zoom

  • Mouse
    PC: Use the mouse wheel to zoom in and out.
    MAC: Use the two-finger up/down gesture on the mouse pad.
  • Keyboard
    Use the [+] and [-] keys to zoom in and out respectively.
    Use the [ALT]+[+] [-] keys to zoom in and out slowly.

Rotate

  • Mouse
    PC: Use the Right drag-click in the left/right directions to rotate the view.
    MAC: Use the two-finger left-right gesture on the mouse pad.
  • Keyboard
    Use the [Shift]+[Left/Right] keys to rotate the view.
    Use the [N] key to return to a "north up" orientation.

Tilt

  • Mouse
    PC: Use the Right drag-click in the up/down directions to rotate the view.
  • Keyboard
    Use the [Shift]+[Up/Down] arrow keys to tilt the view. Use the [R] key to return to a "look down" orientation.
